# Junior Design P2P Payment System Server-Side App

The purpose of our project is to develop a real-time Peer-to-Peer (P2P) payment solution for KPMG. The problem we aim to solve is to provide a convenient and secure way for users to make financial transactions with one another â€“ especially in developing countries.   

## Quick Start Guide
1. Navigate to root of project
2. Run ``npm i`` to install dependencies
3. Run ``npm run build`` to build TS -> JS
4. Run ``npm run start`` to run project
